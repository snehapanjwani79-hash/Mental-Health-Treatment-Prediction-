# Mental Health Treatment Prediction

## 📌 Project Objective
The objective of this project is to identify workplace-related factors that influence whether an employee seeks mental health treatment.

Using real-world survey data from the tech industry, this project analyzes how organizational policies, support systems, and work environment impact treatment-seeking behavior.

---

## 🧠 Problem Statement
Mental health challenges are common in professional environments, but not all employees seek treatment.

This project aims to:

- Identify workplace-related factors affecting treatment decisions
- Analyze patterns through Exploratory Data Analysis (EDA)
- Predict whether an employee is likely to seek mental health treatment

---

## 📊 Dataset Used
OSMI Mental Health in Tech Survey Dataset

---

## ⚙️ Methodology

### 🔹 Data Preprocessing
The following steps were performed:

- Dropped irrelevant / unnecessary columns
- Handled missing values
- Removed duplicate entries
- Encoded categorical variables

---

### 🔹 Exploratory Data Analysis (EDA)
EDA was conducted to understand:

- Workplace support trends
- Treatment-seeking patterns
- Relationship between company policies and mental health decisions

---

### 🔹 Statistical Analysis
The project also included:

- Hypothesis Testing
- Confidence Interval Computation
- Understanding Type I and Type II Errors
- Interpretation of statistical significance

---

## 🤖 Machine Learning Model

Logistic Regression was used for prediction.

This model was chosen because:

- Suitable for binary classification

---

## 📈 Model Performance

Accuracy Achieved:

≈ **0.685**

This indicates the model is reasonably able to capture workplace influence on mental health treatment decisions.
